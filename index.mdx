## Make your first edit

<Steps>
  <Step title="Open this page">
    Edit `index.mdx` in your docs repository.
  </Step>

  <Step title="Start the local preview">
    Run the Mintlify development server from the directory that contains `docs.json`.

    ```bash
    mint dev
    ```
  </Step>

  <Step title="Preview your changes">
    Visit `http://localhost:3000` and refresh the page after saving your edits.
  </Step>
</Steps>
